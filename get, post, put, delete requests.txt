
                                             get, post, put, delete запросы!!!
											
   Метод GET — это самый просто тип HTTP - запроса, которым браузер пользуется каждый раз, когда вы нажимаете
ссылку или вводите URL - адрес в адресною строку. Он инструктирует сервер передать клиенту данные, 
идентифицированные URL - адресом. Никогда не последует изменений данных на стороне сервера в результате запроса GET.
В этом смысле GET запрос доступен только для чтения, но, конечно, как только клиент получит данные, он может самостоятельно 
выполнять любые операции с ними, например, форматировать для изображения.


   Метод PUT - используется, когда вы хотите создать или обновить ресурс, указанный URL - адресом. 
Запросы PUT содержат данные для использования при обновлении или создании ресурса в body. 


   Метод POST - используется, когда обработка, которую вы хотите обработать на сервере, должна повторяться.
Кроме того, POST - запросы должны вызывать обработку body запроса, как подчиненного URL - адреса, который вы отправляете.


   Метод DELETE - должен выполнять противоположное PUT, его следует использовать, если вы хотите удалить ресурс, указанный 
   URL - адресом запроса.
